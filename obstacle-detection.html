Dynamic Obstacle Detection and Avoidance

Human detection and tracking system for an autonomous floor-scrubbing robot using an RGBD camera, deployed on Nvidia Jetson for real-time processing

Project Overview

  • Analysed various human detection methods for an autonomous floor-scrubbing robot
  • Designed, implemented and tested a human tracking algorithm in different lighting and occlusion scenarios utilising an RGBD camera
  • Deployed the project on Nvidia Jetson for real-time processing and execution

Problem Statement

Autonomous cleaning robots operating in public spaces face the challenge of safely navigating around humans while maintaining cleaning efficiency. The system needed to handle various scenarios including poor lighting, partial occlusions, and different human poses and movements.
