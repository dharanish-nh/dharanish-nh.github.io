Improving Autonomous Navigation of Care Robot by Following Social Norms

Autonomous navigation method for the care robot (ROSE) in dynamic indoor environments, ensuring adherence to social norms and respecting the proxemics of individuals

Project Overview

  • Developed an autonomous navigation method for the care robot (ROSE) in dynamic indoor environments, ensuring adherence to social norms and respecting the proxemics of individuals
  • Conducted a comparative analysis of the performance against default navigation using subjective and objective metrics
  • Demonstrated that the improved navigation resulted in enhanced social acceptance

Problem Statement

Traditional robot navigation systems prioritise efficiency and obstacle avoidance but often fail to consider social acceptability. In healthcare environments, robots need to navigate in a manner that feels natural and comfortable to patients, staff, and visitors.

Technical Implementation

Core Technologies

ROS2 Python C++ OpenCV LIDAR Machine Learning Proxemics Theory Social Robotics

Key Features Developed

  • Social-aware path planning algorithms that respect personal space zones
  • Dynamic proxemics modeling based on human behavior patterns
  • Real-time human detection and tracking using LIDAR and camera fusion
  • Adaptive navigation behavior based on environmental context
  • Integration with existing ROS2 navigation stack
  • Machine learning-based prediction of human movement patterns

Novel Contributions
