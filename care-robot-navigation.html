Improving Autonomous Navigation of Care Robot

Enhancing social acceptance through human-aware navigation algorithms that respect personal space and social norms in dynamic healthcare environments

Project Overview

This master's thesis project focused on developing an advanced autonomous navigation system for ROSE (Robot for Social Engagement), a care robot designed to assist in healthcare facilities. The primary challenge was creating a navigation system that not only avoids obstacles but also respects human social norms and personal space preferences.

Problem Statement

Traditional robot navigation systems prioritize efficiency and obstacle avoidance but often fail to consider social acceptability. In healthcare environments, robots need to navigate in a manner that feels natural and comfortable to patients, staff, and visitors.

Technical Implementation

Core Technologies

ROS2 Python C++ OpenCV LIDAR Machine Learning Proxemics Theory Social Robotics

Key Features Developed

  • Social-aware path planning algorithms that respect personal space zones
  • Dynamic proxemics modeling based on human behavior patterns
  • Real-time human detection and tracking using LIDAR and camera fusion
  • Adaptive navigation behavior based on environmental context
  • Integration with existing ROS2 navigation stack
  • Machine learning-based prediction of human movement patterns
